1. mini project 주제
  * 식당에서 테이블 별로 손님의 수와 이용 가격을 계산하는 프로그램.
   
2. 기능
  * CRUD: 테이블 별로 인원수와 가격을 CRUD하는 기능을 가진 함수
  * 파일 저장/읽어오기: 입력받은 정보를 txt파일로 저장, 또는 txt파일에서 정보를 불러오는 기능
  * 검색: 원하는 테이블을 검색해 인원 수와 가격을 확인할 수 있는 기능
